 당신은 농구팀 코치, 지원자 중 한 명을 추가로 선발하고자 함.  
 매 경기에서의 기복이 가장 적은 선수를 선발하자. 어떤 선수를 선발해야 하는가
  
  ```
  import math
candidates = {
  'alan': [8, 14, 6, 8, 14, 9, 14, 9, 15, 5],
  'brad': [11, 4, 11, 7, 9, 7, 8, 7, 6],
  'cate': [16, 22, 15, 12, 3, 20, 17, 13, 23],
  'dave': [24, 15, 18, 18, 12, 9, 19, 23, 13, 14, 18],
}

sd = []

for name, scores in candidates.items():
    # 합계 구하기
    total = sum(scores)
    # value의 전체 개수 구하기
    n = len(scores)
    # 평균구하기
    mean = total / n
    
    # 분산구하기
    vsum = 0
    for x in scores:
      vsum = vsum + (x - mean)**2  # 
    var = vsum / (n-1)
    
# 표준편차 구하기
    std = math.sqrt(var)
    sd.append(std)
    
    
named_sd = list(zip(candidates.keys(), sd))
    
print(named_sd)
```
